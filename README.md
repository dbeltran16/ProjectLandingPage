
# Personal Portfolio Landing Page

This project is a personal portfolio landing page created using **HTML** and **CSS**.  
It introduces who I am and showcases sample projects in a clean, responsive layout.

- HTML
- CSS (Flexbox & Media Queries)

---

##  How to View the Project

### Option 1: Using Live Server (Recommended)
1. Open the project folder in **Visual Studio Code**
2. Install the **Live Server** extension (if not already installed)
3. Right-click on `index.html`
4. Select **"Open with Live Server"**

The page will open automatically in your browser.
(***If you want to be able to toggle between the pages in the navigation bar, 
you will have to download the other .html files)

---

### Option 2: Open Directly in Browser
1. Locate the `index.html` file
2. Double-click the file
3. It will open in your default web browser
